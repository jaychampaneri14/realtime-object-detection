# Realtime Object Detection

> YOLOv8 real-time object detection on webcam/video

## Overview
This project is part of a comprehensive AI/ML portfolio covering computer vision python.

## Features

## Installation

```bash
git clone https://github.com/jaychampaneri14/realtime-object-detection.git
cd realtime-object-detection
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.

```bash
python main.py
```
